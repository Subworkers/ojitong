### workflow 
| 파일        | 역할                                                                                           |
|------------|------------------------------------------------------------------------------------------------|
| main.py    | 프로그램의 주 진입점으로, 각 운영사의 공지사항을 가져오는 함수들을 호출하고, 역 정보 변동을 확인하는 함수를 호출 |
| utils.py   | 공지사항을 가져오고 Slack에 전송하는 함수들이 정의된 파일                                      |
